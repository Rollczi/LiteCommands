### Velocity Extension Dependencies
Maven
```xml
<dependency>
    <groupId>dev.rollczi.litecommands</groupId>
    <artifactId>velocity</artifactId>
    <version>2.2.0</version>
</dependency>
```
Gradle
```groovy
implementation 'dev.rollczi.litecommands:velocity:2.2.0'
```
