### Bukkit Extension Dependencies
Maven
```xml
<dependency>
    <groupId>dev.rollczi.litecommands</groupId>
    <artifactId>bukkit</artifactId>
    <version>2.5.0</version>
</dependency>
```
Gradle
```groovy
implementation 'dev.rollczi.litecommands:bukkit:2.5.0'
```
